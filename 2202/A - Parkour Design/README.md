<div class="header"><div class="title">A. Parkour Design</div><div class="time-limit"><div class="property-title">time limit per test</div>1 second</div><div class="memory-limit"><div class="property-title">memory limit per test</div>256 megabytes</div><div class="input-file input-standard"><div class="property-title">input</div>standard input</div><div class="output-file output-standard"><div class="property-title">output</div>standard output</div></div><div><p>Today, Alex wants to build a parkour course for Steve to train his parkour skills on. A <span class="tex-font-style-it">parkour course</span> is a sequence $$$p_0 \to p_1 \to \ldots \to p_k$$$ of integer coordinates on the plane. Here, a contiguous pair of coordinates is called a <span class="tex-font-style-it">move</span>, denoted as $$$p_{i-1} \to p_i$$$.</p><p>Alex knows that Steve can only perform the following types of moves:</p><ul> <li> $$$(x_i,y_i) \to (x_i+2,y_i+1)$$$; </li><li> $$$(x_i,y_i) \to (x_i+3,y_i)$$$; </li><li> $$$(x_i,y_i) \to (x_i+4,y_i-1)$$$. </li></ul><p>Note that Steve will not perform any other type of moves. For example, Steve can perform $$$(0,0) \to (2,1)$$$ and $$$(2,1) \to (5,1)$$$, but will <span class="tex-font-style-bf">never</span> perform moves such as $$$(2,1) \to (3,2)$$$, $$$(3,0) \to (5,-1)$$$, or $$$(4,-1) \to (6,-1)$$$ (even though they may look very easy).</p><p>You are given an integer coordinate $$$(x,y)$$$ on the plane.</p><p>Please determine if it is possible to make a parkour course $$$q_0,q_1,\ldots,q_k$$$ that satisfies the following conditions:</p><ul> <li> $$$q_0=(0,0)$$$; </li><li> $$$q_k=(x,y)$$$; </li><li> The parkour course <span class="tex-font-style-bf">only consists of</span> moves that Steve can perform. </li></ul></div><div class="input-specification"><div class="section-title">Input</div><p>Each test contains multiple test cases. The first line contains the number of test cases $$$t$$$ ($$$1 \le t \le 10^3$$$). The description of the test cases follows. </p><p>The only line of each test case contains two integers $$$x$$$ and $$$y$$$ ($$$1 \le x \le 10^9$$$, $$$-10^8 \le y \le 10^8$$$).</p></div><div class="output-specification"><div class="section-title">Output</div><p>If it is possible to make a parkour course that satisfies the conditions, output "<span class="tex-font-style-tt">YES</span>" on a separate line.</p><p>If it is <span class="tex-font-style-bf">impossible</span> to make a parkour course that satisfies the conditions, output "<span class="tex-font-style-tt">NO</span>" on a separate line.</p><p>You can output the answer in any case. For example, the strings "<span class="tex-font-style-tt">yEs</span>", "<span class="tex-font-style-tt">yes</span>", and "<span class="tex-font-style-tt">Yes</span>" will also be recognized as
