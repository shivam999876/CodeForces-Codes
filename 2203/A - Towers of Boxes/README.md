The weight of each box is $$$m$$$, and the durability of each box is $$$d$$$.</p><p>To save space, Monocarp wants to build several "towers" of boxes by stacking them on top of each other. Each tower will consist of a positive (greater than $$$0$$$) integer number of boxes stacked on top of each other. To ensure that no box breaks, the following condition must be met:</p><ul> <li> for each box, the <span class="tex-font-style-bf">total</span> weight of all boxes above it must not exceed the durability of that box. </li></ul><p>Help Monocarp calculate the minimum number of towers he can achieve, given that each of the $$$n$$$ boxes must be used.</p></div><div
